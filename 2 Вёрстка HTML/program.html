Описание курса

В первой части курса мы познакомились со стандартами Web 2.0, научились работать с макетами и форматировать код, освоили адаптивную верстку, а по итогу - можем создавать быстрые и удобные сайты, которые точно понравятся пользователям.

Мы поняли назнаение основных тегов HTML.,можем без усилийстилизовать и размещать элементы с помощью CSS и сверстали свой первый макет.

Вторая часть курса сделает из вас настоящего мастера. Вы изучите сборщики проектов, научитесь верстать email-рассылки, разберетесь с анимацией на сайте.

Помимо этого, вы сверстаете фрагмент из интернет-магазина - главную страницу каталога и страницу товара!
